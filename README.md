# 福爾摩沙不惑之聲《走找青春》演後票選 × 好禮抽獎活動執行包

合唱團音樂會演後互動活動的完整執行工具包：聽眾票選最喜歡的兩首曲目（排序）、抽 5 位送音樂會入場券或 30週年紀念 USB（二選一），終極目標是把演後熱度轉化為[粉絲專頁](https://www.facebook.com/a4ssingers)追蹤數成長。

## 給小編：怎麼用

1. 先讀 `執行包/00_總覽與時程.md`（總指揮：逐日時程、角色分工、前置檢查、KPI）
2. 建置日照 `08_自動化使用說明.md` 安裝 Apps Script（`07_自動化程式.gs`），表單建立／自動關閉／統計／抽獎名單全部一鍵完成
3. 文案直接複製：FB 貼文在 `03`、LINE 訊息（14 則，含開場公告訊息 0）在 `05`
4. 開獎與領獎照 `04` 的 SOP
